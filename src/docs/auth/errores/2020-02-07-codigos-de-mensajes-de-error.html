Códigos de mensajes de error

En esta página encontrará información de ayuda complementaria sobre el significado de los códigos de mensaje de error y qué se debe hacer para solucionarlos.

La solución a los códigos de error depende de los últimos dos dígitos del código.

Ejemplo: Para el código 01.01.123.212.95.28. debe buscar en la columna izquierda el número 28 y en la celda del lado derecho encontrará la información de ayuda.
